sublime-text-3-settings

Sublime Text 3 User settings


*Where to put files*

* Linux: ~/.config/sublime-text-3/Packages/User
* OS X: ~/Library/Application Support/Sublime Text 3/Packages/User
* Windows: %APPDATA%\Sublime Text 3\Packages\User
